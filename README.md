# Flutter Facebook Clone. 

Flutter Facebook Clone. 
This is a simple flutter application where I have clonned Facebook UI.
Created by [Joan Louji](https://joanlouji.web.app/)

## Installation

``` 

git clone https://github.com/sjlouji/FacebookClone-Flutter.git 
flutter pub get
flutter run 

```
